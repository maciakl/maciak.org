Fix My Printer Print Spooler Cleanup

Description

Having trouble printing? Documents piling up in the printer queue? Can't delete failed print jobs using the built in printer application?

It sounds like something is stuck in your printer queue. Fix My Printer is a tool designed to fix that very problem

Technical Information

This tool does not touch printer drivers, or diagnose printer connectivity issues. It shuts down the spooler service, deletes all the files from the spool folder and restarts the service.

    net stop spooler
    del %SystemRoot%\SYSTEM32\SPOOL\PRINTERS\*
    net start spooler
