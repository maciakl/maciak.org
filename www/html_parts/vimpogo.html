Vim 7.3 For Pogo Plug

Vim for Pogo Plug

One of the first things I did after I set up my $30 Linux server on a Pogo Plug was to compile Vim 7.3. I figured I might as well share the compiled deb packages here in case someone wants to use them on their plug.

The packages below were compiled for Debian 6.0.6 (squeeze) running the 2.6.32-5-kirkwood armv5tel kernel. If you were following my setup instructions for setting up the plug, they should work flawlessly.
