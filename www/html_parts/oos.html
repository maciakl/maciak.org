Open Source Projects

Here are some links to some of the open source projects I created or contribute to:

  • Plain Text Journal - a simple personal journal/diary tool.
  • My Temporary Public Key - Easy to use, public key encryption tool for end-users.
  • Student Webspace for Google App Engine - a Google Engine App that allows users to host small html webpages. Uses Blobstore and Datastore features.
  • PyTris - a Tetris clone written in Python and PyGame
  • PerfMonG - a simple performance monitoring tool.
  • Twimi - a command line twitter client for linux.
  • SITS - a php based issue tracking system
  • JTwitt - twitter framework written in Java
