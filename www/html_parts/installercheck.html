Installer Check Command Line Tool

Description

Command line tool which tells you the size of your Installer folder. Useful to diagnose disk space issues. It is not uncommon for Windows Update to repeatedly fail installing an update, and leave redundant copies of the installation file in the Installer folder until the machine runs out of disk space. This tool lets you quickly check the size of that folder.

To use simply double-click or run from the command line.
