Description

Hypothetical scenario: you have a folder of 30-40 images that you want to upload somewhere online. You don't want to use a fancy image sharing service. You don't want lightbox effects or AJAX driven gallery. All you want to do is create thumbnails for all your files and generate simple HTML gallery to display them.

Simple Gallery Maker is a tool that will do exactly that. You supply a folder full of images, choose the thumbnail size and it will do the rest. When it is done, you will have a thumbnail for each image and a clean HTML file with a simple gallery 4 image wide. Maximum simplicity, minimum fuss.

Note: Simple Gallery Maker will ignore image files with names starting with tmb_. This prefix is used for generated thumbnails.
