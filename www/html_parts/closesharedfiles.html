Close Shared Files Disconnect users from network files

Description

Has this ever happened to you?

You try to open a document located on a network share and it is in Read-Only mode. No one knows who has it open. What do you do?

It's simple, you disconnect it using openfiles.exe command. Close Shared Files is a graphical wrapper around that utility. Type the name of the server into the text box to see shared files that have open connections. Please note you will need to have permissions to disconnect files on that server.
