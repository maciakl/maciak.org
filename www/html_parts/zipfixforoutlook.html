Zip Fix for Outlook

Description

Trying to attach a zip file to your email in Outlook but the dialog box shows none of them? Did your Outlook start hiding all the zip files for absolutely no reason?

This is a common Outlook 2003 issue caused by the way Windows XP handles zip files as folders. This tool will fix this issue for you allowing Outlook to see the zip files again.

If you are having issues, run the fix and then restore to default settings. That may clear things up too.

Technical Information

Here is what this tool actually does:

  • The Fix Zip File Issue button runs regsvr32 /u zipfldr.dll
  • The Restore to default Settings button runs regsvr32 zipfldr.dll
