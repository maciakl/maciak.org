Tools I Made

  • jjmp - bookmarks for the CLI.
  • gopher - an opinionated Golang project bootstrapping tool.
  • weather - weather forecast on the command line.
  • os - a cross platform os detection tool.
  • pv - a previewer for LF file manager.
  • myip - display your IP address.
  • dashword - simple password generator.
  • post - http post a file to an upload site.
  • sieve - filter text files.
  • netsquirt - serve a single file over http.
  • createfile - create a file of arbitrary size.
  • dicword - randomly generate a dictionary word.
